# Ua

Judge the client-side userAgent. These are class static methods.

# Get Class

const { Ua } = bekit.helper;

# Static Methods

# Ua.isWx()

  • Returns: {boolean}
  • Detail: If it's Wechat

# Ua.isIOS()

  • Returns: {boolean}
  • Detail: If it's an IOS device

# Ua.isAndroid()

  • Returns: {boolean}
  • Detail: If is an Android device

# Ua.isMob()

  • Returns: {boolean}
  • Detail: If is an mobile device
