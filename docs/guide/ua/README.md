# Ua
判断客户端环境，方法为 Ua类的静态方法。

## 获取类
```javascript
const { Ua } = bekit.helper;
```

## 静态方法
### Ua.isWx()
- 返回值：{boolean}
- 详细：是否为微信客户端

### Ua.isIOS()
- 返回值：{boolean}
- 详细：是否为 IOS 设备

### Ua.isAndroid()
- 返回值：{boolean}
- 详细：是否为 Android 设备

### Ua.isMob()
- 返回值：{boolean}
- 详细：是否为移动端
